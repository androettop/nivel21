Atajos para Tokens
Controla rápidamente los tokens/fichas del mapa usando atajos de teclado. Selecciona un token primero.
H Alternar visibilidad del token
B Bloquear/desbloquear el token
E Editar el token seleccionado
D Duplicar el token seleccionado
Delete Eliminar el token seleccionado
Orden de Altura de Tokens
Ajusta la altura de uno o varios tokens seleccionados para colocarlos encima o debajo de otros en el orden deseado.
Pg Up Subir altura del token (+1)
Pg Down Bajar altura del token (-1)
